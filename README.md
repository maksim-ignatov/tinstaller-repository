## FAQ

**Вопрос: У меня не качаются приложения, появляется восклицательный знак! Что делать?**

Ответ: Нужно полностью удалить папку `ti` в `Downloads` или `Загрузки` на Android устройстве и перезагрузить приложение `TInstaller`.
